# Session Briefing

Provide a quick orientation for the current coding session.

1. Run `git status` to show current branch and any uncommitted changes.
2. Run `git log --oneline -10` to show the last 10 commits.
3. Run `git diff --stat` to summarize any staged/unstaged changes.
4. Summarize:
   - Current branch and whether it's clean or has pending work
   - What the recent commits were about (group by feature/area)
   - Any uncommitted changes that need attention
   - Suggested next steps based on what you see

Keep the summary concise — aim for a quick "here's where you are" in under 10 lines.
