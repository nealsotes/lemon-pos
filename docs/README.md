# QuickServe Documentation

This folder contains all project documentation and sub-agent guidelines for the QuickServe POS system.

## Documentation Structure

### Main Documentation
- **AGENTS.md** - Main project documentation (if exists)

### Sub-Agent Documentation
- **BACKEND_AGENT.md** - .NET 8 Web API backend development guidelines
- **FRONTEND_AGENT.md** - Angular 20 frontend development guidelines
- **DATABASE_AGENT.md** - MySQL database operations and migrations
- **DEVOPS_AGENT.md** - Docker and Railway deployment guidelines
- **TESTING_AGENT.md** - Testing strategies and patterns
- **TEAM_LEAD_AGENT.md** - Project management and team leadership

### Guides
- **SUB_AGENTS_README.md** - Overview of all sub-agents
- **HOW_TO_USE_SUB_AGENTS.md** - Practical guide on using sub-agents

## Quick Start

1. **New to the project?** Start with `SUB_AGENTS_README.md` for an overview
2. **Working on backend?** See `BACKEND_AGENT.md`
3. **Working on frontend?** See `FRONTEND_AGENT.md`
4. **Managing the team?** See `TEAM_LEAD_AGENT.md`
5. **Need help using these docs?** See `HOW_TO_USE_SUB_AGENTS.md`

## File Locations

All documentation files are now located in the `docs/` folder at the project root. When referencing these files, use the `docs/` prefix:

- `docs/BACKEND_AGENT.md`
- `docs/FRONTEND_AGENT.md`
- etc.
